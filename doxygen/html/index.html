Pilha de Vetor de ItemType

Modo de Uso :

ItemType v[3];
pilha p;
p = vetorParaPilha(v, 3);

Também é possível utilizar facilmente valores unitários

ItemType v[] = {1, 2, 3};
pilha p;
p = vetorParaPilha(v, 3); //(Vetor de ItemType, Número de elementos)
